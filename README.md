Dataset Putusan Perkara Narkotika di Pengadilan Negeri Surabaya
Selamat datang di repository ini! Di dalamnya, terdapat dataset lengkap berisi hasil putusan perkara narkotika yang diadili di Pengadilan Negeri Surabaya. Dataset ini dikumpulkan untuk mendukung penelitian, analisis, dan pengembangan sistem yang berhubungan dengan prosedur peradilan pidana narkotika di Indonesia.

Gambaran Umum Dataset
Dataset ini terdiri dari ratusan entri yang berisi informasi detail tentang setiap kasus narkotika yang diputuskan, mulai dari identifikasi kasus, barang bukti, hingga isi putusan pengadilan. Data ini sangat penting untuk memahami pola umum penanganan kasus narkotika, termasuk jenis barang bukti yang paling umum ditemukan, durasi hukuman, serta tindakan pengadilan terhadap barang bukti seperti dimusnahkan atau disita.

Struktur Dataset
Setiap baris data merepresentasikan satu putusan pengadilan dan terdiri dari kolom-kolom berikut:

NO: Nomor urut data.

NO PUTUSAN: Nomor registrasi atau nomor resmi dari putusan.

LEMBAGA PERADILAN: Nama pengadilan yang menangani kasus, umumnya Pengadilan Negeri Surabaya.

BARANG BUKTI: Rincian barang bukti yang ikut disita, termasuk jenis narkotika (sabu, ganja, ekstasi) dan beratnya. Juga mencakup nomor registrasi laboratorium forensik jika ada.

AMAR PUTUSAN: Ringkasan keputusan hakim, yang menyatakan terdakwa bersalah, jenis tindak pidana, hukuman penjara, besaran denda, serta tindakan terhadap barang bukti.

Kenapa Dataset Ini Penting?
Data ini memberikan wawasan nyata tentang bagaimana sistem peradilan negara menangani kasus narkotika. Dengan menganalisis data ini, kita dapat mengidentifikasi tren, pola, dan kelemahan dalam proses penegakan hukum narkotika di Indonesia. Selain itu, dataset ini dapat membantu akademisi, peneliti, dan pengambil kebijakan dalam merumuskan strategi pengendalian narkotika yang lebih efektif.

Potensi Penggunaan Data
Analisis Statistik: Menggali pola hukuman dan jenis barang bukti.

Kajian Hukum: Meneliti konsistensi hukuman terhadap jenis narkotika tertentu.

Pengembangan Model: Membuat model prediktif untuk perkiraan hukuman berdasarkan barang bukti dan faktor lain.

Literatur dan Penelitian: Menjadi bahan sumber referensi dalam penelitian tentang kriminalitas narkotika.

Cara Menggunakan Dataset
Anda dapat mengakses file lengkap dataset ini dan menggunakannya sesuai kebutuhan, baik untuk analisis data, pengembangan sistem, maupun studi akademik. Dataset ini juga siap untuk diintegrasikan ke dalam proyek machine learning, terutama dalam bidang klasifikasi kasus pidana maupun analisis kriminalitas.
