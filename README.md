# Dataset Putusan Perkara Narkotika di Pengadilan Negeri Surabaya

Selamat datang di repository dataset putusan perkara pidana narkotika yang berasal dari Pengadilan Negeri Surabaya. Dataset ini berisi informasi komprehensif terkait putusan pengadilan atas kasus narkotika selama beberapa waktu terakhir. Tujuan utama dari dataset ini adalah untuk memberikan dasar data yang solid bagi peneliti, praktisi hukum, dan pengambil kebijakan dalam memahami penanganan kasus narkotika di Indonesia.

---

## Tentang Dataset Ini

Dataset ini menyajikan data putusan perkara narkotika yang mencakup berbagai aspek penting, termasuk identitas perkara, jenis dan berat barang bukti, serta isi amar putusan pengadilan. Setiap baris data mewakili satu putusan yang terdiri dari:

- **NO**: Nomor urut data putusan.
- **NO PUTUSAN**: Nomor resmi putusan perkara.
- **LEMBAGA PERADILAN**: Nama pengadilan yang menangani perkara, sebagian besar adalah Pengadilan Negeri Surabaya.
- **BARANG BUKTI**: Rincian barang bukti yang berkaitan dengan narkotika (misalnya sabu, ganja, ekstasi) lengkap dengan berat bersih dan nomor registrasi laboratorium forensik.
- **AMAR PUTUSAN**: Tulisan keputusan hakim yang secara rinci menjelaskan terdakwa, hasil dakwaan, pidana penjara dan denda yang dijatuhkan, serta penetapan terhadap barang bukti.

Data ini sangat berguna untuk memahami bagaimana sistem peradilan menangani narkotika, serta memberikan gambaran pola hukuman yang dijatuhkan berdasarkan jenis dan berat barang bukti.

---

## Manfaat dan Potensi Penggunaan

- **Analisis Hukum dan Kriminalitas:** Memahami distribusi jenis narkotika dan hukuman terkait.
- **Riset Akademik:** Sebagai dataset untuk studi empiris di bidang hukum pidana dan kebijakan narkotika.
- **Pengembangan Model Prediktif:** Membantu dalam pembuatan model machine learning yang dapat mengklasifikasikan dan memprediksi putusan berdasarkan data kasus.
- **Evaluasi Proses Peradilan:** Menganalisis konsistensi dan keadilan dalam penjatuhan hukuman.

---

## Struktur Data

| Kolom           | Deskripsi                                                                                      |
|-----------------|-----------------------------------------------------------------------------------------------|
| NO              | Nomor urut entri                                                                              |
| NO PUTUSAN      | Nomor resmi putusan pengadilan                                                                |
| LEMBAGA PERADILAN| Nama Pengadilan (misal: PN Surabaya)                                                        |
| BARANG BUKTI    | Rincian barang bukti: jenis narkotika, berat netto, dan catatan terkait                        |
| AMAR PUTUSAN    | Teks lengkap amar putusan yang berisi keputusan hakim, hukuman, denda, dan pengelolaan bukti  |

## Cara Menggunakan Dataset

1. **Download** file dataset dari repository.
2. Gunakan perangkat lunak spreadsheet atau bahasa pemrograman seperti Python (pandas) untuk membuka dan mengolah data.
3. Terapkan analisis statistik, visualisasi, atau machine learning sesuai kebutuhan.
4. Manfaatkan kolom `AMAR PUTUSAN` untuk text mining jika berminat mendalami isi putusan.
